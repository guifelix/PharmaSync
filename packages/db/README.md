# Database Package

Phase 1 uses AdonisJS Lucid migrations, models, factories, and seeders in `apps/api`.

This package is reserved for shared SQL fixtures, public data import notes, and database utilities that must be reused outside the Adonis app.

Schema conventions, tenant scoping rules, index strategy, and the current table-level model live in [docs/system-design/database-design.md](../../docs/system-design/database-design.md).
